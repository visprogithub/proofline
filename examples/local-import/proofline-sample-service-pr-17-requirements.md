# Evidence export requirements

## REQ-101: Export evidence reports

- Reviewers can download Markdown and JSON evidence reports.

## REQ-102: Clear analysis data

- Starting a new case removes the current analysis from page memory.

## REQ-103: Reject oversized imports

- Files above the configured import limit fail with clear guidance.

## REQ-104: Announce analysis status

- Screen readers receive progress and completion announcements.

## REQ-105: Preserve reviewer filters

- Evidence filters remain selected while the current analysis is open.
